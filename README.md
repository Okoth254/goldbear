# Goldbear Storefront & Medusa Admin Backend

This repository contains the Flutter application for the Goldbear Storefront, alongside the Medusa Backend and Medusa Admin dashboard.

## Quickstart

Follow our [quickstart guide](https://docs.medusajs.com/quickstart/quick-start) to learn how to set up a Medusa server.

## Setting up Admin

1. **Install dependencies**
   ```
   yarn install
   ```
2. **Start the development server**
   ```
   yarn dev
   ```
3. **Go to [http://localhost:7000](http://localhost:7000)**
